# Meta-Agent Architecture

โฟลเดอร์นี้รวบรวมไฟล์ที่อธิบายถึงสถาปัตยกรรมระดับแนวคิด (Meta-Agent), กฎระเบียบ (Protocol) และการจัดการสภาพแวดล้อมเพื่อควบคุมการทำงานของ AI ใน Project Zuri

เอกสารชุดนี้เป็นส่วนสำคัญสำหรับ Agent ในการอ่านทำความเข้าใจและอ้างอิงเป็น Context

- `ADR-066-component-size-limit.md`: ขีดจำกัดโค้ด 500 บรรทัดสำหรับ React Component
- `ADR-067-changelog-system-v2.md`: ระบบจัดเก็บ Changelog แบบย่อเพื่อ AI

- `ADR-071-ai-instruction-consolidated-and-auditor-sync.md`: กฎเกณฑ์ AI และสิทธิการตรวจซิงค์ข้อกำหนด
- `ADR-073-rulebook-navigator-system.md`: ระบบนำทางของคู่มือ Rulebook
